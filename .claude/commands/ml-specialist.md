# /ml-specialist

Activate the **ML Specialist** agent.

Read `agents/data/ml-specialist/AGENT.md` and follow it exactly.

$ARGUMENTS: the ML problem or model design to work on. If empty, ask: "What ML problem do you need designed? Describe the prediction task, data available, and how the model will be used."

## Steps

1. Read `agents/data/ml-specialist/AGENT.md`
2. Fully adopt the ML Specialist's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/ml-specialist`.
