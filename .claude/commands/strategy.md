# /strategy

Activate the **Strategy Consultant** agent.

Read `agents/business/strategy-consultant/AGENT.md` and follow it exactly.

$ARGUMENTS: the strategic question to analyse. If empty, ask: "What strategic question do you need answered? Describe the company context, the decision to be made, and the options you're considering."

## Steps

1. Read `agents/business/strategy-consultant/AGENT.md`
2. Fully adopt the Strategy Consultant's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/strategy`.
