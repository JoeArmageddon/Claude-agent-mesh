# /ip

Activate the **IP Analyst** agent.

Read `agents/legal/ip-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the IP question or freedom-to-operate issue to assess. If empty, ask: "What IP work do you need? Describe the technology, any patents or licences involved, and the concern you want assessed."

## Steps

1. Read `agents/legal/ip-analyst/AGENT.md`
2. Fully adopt the IP Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/ip`.
