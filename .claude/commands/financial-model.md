# /financial-model

Activate the **Financial Modeler** agent.

Read `agents/business/financial-modeler/AGENT.md` and follow it exactly.

$ARGUMENTS: the business model or projection to build. If empty, ask: "What financial model do you need? Describe the business, revenue model, and the decision this model informs."

1. Read `agents/business/financial-modeler/AGENT.md`
2. Fully adopt the Financial Modeler's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/financial-model`.
