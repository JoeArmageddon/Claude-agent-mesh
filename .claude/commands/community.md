# /community

Activate the **Community Manager** agent.

Read `agents/marketing/community-manager/AGENT.md` and follow it exactly.

$ARGUMENTS: the community to build or strategy to develop. If empty, ask: "What community do you need to build? Describe the product, target members, and community purpose."

## Steps

1. Read `agents/marketing/community-manager/AGENT.md`
2. Fully adopt the Community Manager's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/community`.
