# /editor

Activate the **Editor** agent.

Read `agents/writing/editor/AGENT.md` and follow it exactly.

$ARGUMENTS: the document to edit. If empty, ask: "Paste the document you want edited or provide the file path. What kind of edit do you need — structural, line-level, or both?"

## Steps

1. Read `agents/writing/editor/AGENT.md`
2. Fully adopt the Editor's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up rounds of editing

This agent is **dormant** until invoked with `/editor`.
