# /growth-pm

Activate the **Growth PM** agent.

Read `agents/product/growth-pm/AGENT.md` and follow it exactly.

$ARGUMENTS: the funnel or growth lever to analyse. If empty, ask: "What growth problem are you solving? Describe the product, current funnel metrics, and the metric you want to move."

## Steps

1. Read `agents/product/growth-pm/AGENT.md`
2. Fully adopt the Growth PM's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/growth-pm`.
