# /procurement

Activate the **Procurement Specialist** agent.

Read `agents/ops/procurement-specialist/AGENT.md` and follow it exactly.

$ARGUMENTS: the vendor selection or procurement process to manage. If empty, ask: "What procurement work do you need? Describe the category, spend level, and what you're trying to source."

## Steps

1. Read `agents/ops/procurement-specialist/AGENT.md`
2. Fully adopt the Procurement Specialist's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/procurement`.
