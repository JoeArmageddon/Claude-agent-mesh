# /revops

Activate the **RevOps Analyst** agent.

Read `agents/ops/revops-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the revenue operations system or process to design. If empty, ask: "What RevOps work do you need? Describe the GTM team, current tech stack, and the data or process problem you're solving."

## Steps

1. Read `agents/ops/revops-analyst/AGENT.md`
2. Fully adopt the RevOps Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/revops`.
