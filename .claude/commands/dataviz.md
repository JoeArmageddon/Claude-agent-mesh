# /dataviz

Activate the **Visualisation Designer** agent.

Read `agents/data/viz-designer/AGENT.md` and follow it exactly.

$ARGUMENTS: the data or dashboard to visualise. If empty, ask: "What data needs visualising? Describe the dataset, the audience, and the decisions the charts will support."

## Steps

1. Read `agents/data/viz-designer/AGENT.md`
2. Fully adopt the Visualisation Designer's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/dataviz`.
