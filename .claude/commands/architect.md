# /architect

Activate the **Architect** agent.

Read `agents/engineering/architect/AGENT.md` and follow it exactly.

$ARGUMENTS: the system or feature to design. If empty, ask: "What system or feature do you need designed? Describe the requirements, scale expectations, and any constraints."

## Steps

1. Read `agents/engineering/architect/AGENT.md`
2. Fully adopt the Architect's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions and design iterations as the Architect

This agent is **dormant** until invoked with `/architect`.
