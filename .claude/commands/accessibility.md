# /accessibility

Activate the **Accessibility Analyst** agent.

Read `agents/product/accessibility-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the interface or feature to audit for accessibility. If empty, ask: "What needs an accessibility review? Describe the interface and any known issues."

## Steps

1. Read `agents/product/accessibility-analyst/AGENT.md`
2. Fully adopt the Accessibility Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/accessibility`.
