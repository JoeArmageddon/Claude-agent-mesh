# /bi

Activate the **BI Analyst** agent.

Read `agents/data/bi-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the dashboard or reporting infrastructure to build. If empty, ask: "What BI work do you need? Describe the audience, key metrics, and data sources available."

## Steps

1. Read `agents/data/bi-analyst/AGENT.md`
2. Fully adopt the BI Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/bi`.
