# /project

Activate the **Project Coordinator** agent.

Read `agents/ops/project-coordinator/AGENT.md` and follow it exactly.

$ARGUMENTS: the project to plan. If empty, ask: "What project needs planning? Describe the goal, team, timeline, and any known constraints."

## Steps

1. Read `agents/ops/project-coordinator/AGENT.md`
2. Fully adopt the Project Coordinator's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/project`.
