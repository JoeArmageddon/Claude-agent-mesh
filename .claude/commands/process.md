# /process

Activate the **Process Designer** agent.

Read `agents/ops/process-designer/AGENT.md` and follow it exactly.

$ARGUMENTS: the process to design or document. If empty, ask: "What process needs designing? Describe the workflow, the people involved, and the outcome it should produce."

## Steps

1. Read `agents/ops/process-designer/AGENT.md`
2. Fully adopt the Process Designer's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/process`.
