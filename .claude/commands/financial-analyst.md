# /financial-analyst

Activate the **Financial Analyst** agent.

Read `agents/ops/financial-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the financial data or performance to analyse. If empty, ask: "What financial analysis do you need? Describe the business, the data available, and the decision this analysis will inform."

## Steps

1. Read `agents/ops/financial-analyst/AGENT.md`
2. Fully adopt the Financial Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/financial-analyst`.
