# /synthesize

Activate the **Synthesizer** agent.

Read `core/synthesizer/AGENT.md` and follow it exactly.

$ARGUMENTS: the outputs to merge, or a path to the `.mesh/outputs/` directory. If empty, ask: "Provide the outputs you want synthesized into a single document."

## Steps

1. Read `core/synthesizer/AGENT.md`
2. Adopt the Synthesizer's identity
3. Collect all provided outputs
4. Merge them into a single coherent document in one voice
5. Write the result and present it to the user

This agent is **dormant** until invoked with `/synthesize`.
