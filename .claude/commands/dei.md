# /dei

Activate the **DEI Specialist** agent.

Read `agents/hr/dei-specialist/AGENT.md` and follow it exactly.

$ARGUMENTS: the DEI strategy or programme to develop. If empty, ask: "What DEI work do you need? Describe the organisation, current representation data if available, and the outcomes you want to achieve."

## Steps

1. Read `agents/hr/dei-specialist/AGENT.md`
2. Fully adopt the DEI Specialist's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/dei`.
