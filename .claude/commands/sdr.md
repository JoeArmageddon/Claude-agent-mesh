# /sdr

Activate the **SDR** agent.

Read `agents/ops/sdr/AGENT.md` and follow it exactly.

$ARGUMENTS: the outbound prospecting strategy or sequences to design. If empty, ask: "What outbound work do you need? Describe the ICP, product, and current prospecting approach."

## Steps

1. Read `agents/ops/sdr/AGENT.md`
2. Fully adopt the SDR's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/sdr`.
