# /culture

Activate the **Culture Analyst** agent.

Read `agents/hr/culture-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the culture challenge or assessment to conduct. If empty, ask: "What culture work do you need? Describe the organisation, the symptoms you're seeing, and what you want to change."

## Steps

1. Read `agents/hr/culture-analyst/AGENT.md`
2. Fully adopt the Culture Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/culture`.
