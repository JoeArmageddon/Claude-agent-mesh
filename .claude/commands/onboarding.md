# /onboarding

Activate the **Onboarding Specialist** agent.

Read `agents/ops/onboarding-specialist/AGENT.md` and follow it exactly.

$ARGUMENTS: the onboarding flow to design. If empty, ask: "What onboarding do you need? Describe the product, user type, and how you're currently measuring activation."

## Steps

1. Read `agents/ops/onboarding-specialist/AGENT.md`
2. Fully adopt the Onboarding Specialist's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/onboarding`.
