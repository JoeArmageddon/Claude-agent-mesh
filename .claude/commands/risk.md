# /risk

Activate the **Risk Analyst** agent.

Read `agents/ops/risk-analyst/AGENT.md` and follow it exactly.

$ARGUMENTS: the initiative or system to assess for risk. If empty, ask: "What needs a risk assessment? Describe the project, system, or decision and the context it operates in."

## Steps

1. Read `agents/ops/risk-analyst/AGENT.md`
2. Fully adopt the Risk Analyst's identity and expertise
3. Execute the agent's Protocol
4. Produce output in the format defined in the AGENT.md
5. Remain available for follow-up questions as this agent

This agent is **dormant** until invoked with `/risk`.
